ISEFY Website Asset Requirements List (Detailed)
==============================================
Project Name: ISEFY Security Products Website
Design Theme: High-Tech, Trusted, Modern, Clean.
Brand Colors (Reference):
- Primary Blue: #0D47A1
- Dark Blue: #002171
- Primary Purple: #7B1FA2
- Accent Cyan: #00E5FF

Instructions for Design Team:
- All images should be optimized for web (72 DPI).
- Please strictly adhere to the suggested dimensions/ratios to fit the coded layout.
- "Safe Area": Ensure main subject is centered for responsive cropping.

1. Brand Assets
---------------
[ ] **Logo**
    - **Format**: PNG (Transparent Background) or SVG (Vector preferred)
    - **Dimensions**: Min height 100px.
    - **Variations**: 
        1. Full Color (Dark Text) for white backgrounds.
        2. All White (Light Text) for dark/footer backgrounds.

2. Home Page Assets
-------------------
[ ] **Hero Section Background**
    - **Context**: Main landing banner, behind text "Complete Home Protection".
    - **Dimensions**: 1920 x 1080 px (16:9 Aspect Ratio).
    - **Style**: Dark, atmospheric, high-tech. Abstract pattern or subtle smart city/home environment.
    - **Note**: Image will be overlaid with a dark blue opacity layer (60-80%), so high contrast elements work best.

[ ] **Ecosystem Section Background**
    - **Context**: Mid-page section "Complete Home Protection".
    - **Dimensions**: 1920 x 1080 px (16:9 Aspect Ratio).
    - **Style**: Abstract connectivity, digital waves, or network nodes. Dark tech theme.

[ ] **Core Capabilities Feature Image**
    - **Context**: "Core Capabilities" section, side-by-side with feature list.
    - **Dimensions**: 1200 x 1200 px (1:1 Square Aspect Ratio).
    - **Style**: High-tech close-up of a camera lens, chip, or AI visualization. Clean and futuristic.

[ ] **Cloud Services Image**
    - **Context**: "ISEFY Protect" Cloud section.
    - **Dimensions**: 800 x 600 px (4:3 Aspect Ratio).
    - **Style**: Illustration or photo representing cloud security, data shield, or mobile app data view. Light/Clean theme (white/blue background).
